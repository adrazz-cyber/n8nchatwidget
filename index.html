n8n Chat Widget Demo

This page demonstrates a customizable chat widget that connects to n8n webhooks.

Widget Configuration

The chat widget is configured with the following settings:

    webhook: {
        url: 'YOUR_N8N_WEBHOOK_URL',
        route: 'general'
    },
    branding: {
        logo: 'https://seoitright.co.uk/wp-content/uploads/2024/12/SEO-it-Right-02-scaled.jpg',

        responseTimeText: 'We typically respond right away'
    },
    style: {
        primaryColor: '#3da5ae',
        secondaryColor: '#001d4c',
        position: 'right',
        backgroundColor: '#ffffff',
        fontColor: '#333333'
    }
};

Installation

To add this chat widget to your website, simply include the following code:

<!-- Widget Configuration -->
<script>
    window.ChatWidgetConfig = {
        webhook: {
            url: 'YOUR_N8N_WEBHOOK_URL',
            route: 'general'
        },
        branding: {
            logo: 'YOUR_LOGO_URL',
            name: 'YOUR_COMPANY_NAME',
            welcomeText: 'YOUR_WELCOME_MESSAGE',
            responseTimeText: 'YOUR_RESPONSE_TIME_TEXT'
        },
        style: {
            primaryColor: '#YOUR_PRIMARY_COLOR',
            secondaryColor: '#YOUR_SECONDARY_COLOR',
            position: 'right', // or 'left'
            backgroundColor: '#ffffff',
            fontColor: '#333333'
        }
    };
</script>
<script src="https://cdn.jsdelivr.net/gh/adrazz-cyber/n8nchatwidget@main/chat-widget.js"></script>

Important Notes:

  • Make sure to replace YOUR_N8N_WEBHOOK_URL with your actual n8n webhook URL.
  • The webhook response should be in the format { "message": "Response text" }.
  • Configure your n8n workflow to accept a POST request with the JSON payload.
