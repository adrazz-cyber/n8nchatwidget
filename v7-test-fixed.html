This page demonstrates the enhanced SEO it Right chat widget (v7). The widget includes:

  • Professional styling with SEO it Right branding colors

  • Smooth animations and transitions
  • Typing indicators
  • Integration with n8n webhooks

Click the Chat with us button in the bottom right corner to test the widget.

Implementation Guide

To add this chat widget to your website, simply include the following script tag before the closing </body> tag:

<script src="https://cdn.jsdelivr.net/gh/adrazz-cyber/n8nchatwidget@main/chat-widget-v7.js"></script>

Important: Before using on your website, you must open the script file and replace the placeholder webhook URL with your actual n8n webhook URL.

The widget is fully customizable by editing the configuration at the top of the JavaScript file.

Configuration Options

You can customize the widget by editing these settings in the JavaScript file:

  • webhook.url - Your n8n webhook URL
  • webhook.responseField - Field that contains the bot response
  • branding - Title, subtitle, logo, and name settings
  • style - Colors, fonts, sizes, and positioning
  • initialMessages - Messages shown when the chat first opens
